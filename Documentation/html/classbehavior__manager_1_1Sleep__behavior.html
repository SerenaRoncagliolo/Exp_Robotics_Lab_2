Detailed Description

class Sleep_behavior

This class implement the SLEEP behaviour of the robot pet The robot sleeps (SLEEP state)for a random period of time, then it moves to NORMAL state The robot should:

  • reach a predefined location within the arena in Gazebo
  • stays there for some times
  • goes back in NORMAL state
